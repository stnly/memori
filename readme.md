#Memori

Memori is a minimal note/thought taking cli tool.

##How it works

    Usage: m [-a] [-h|--help] [-l <number>] <message>

    Current file path: $mPath

       -a                   Lists all memories
       -h, --help           Shows this
       -l                   List last 10 lines of memories
       -l <number>          List last <number> of memories
